# XBrainLab

**EEG Analysis Platform with AI Agent**

XBrainLab is a comprehensive platform for EEG (electroencephalography) data analysis,
featuring an integrated AI agent for guided experiment workflows.

## Features

- **Data Loading** — Import EEG data from multiple formats (EDF, BDF, MAT, etc.)
- **Preprocessing** — Apply bandpass filtering, ICA, channel selection, and more
- **Dataset Splitting** — K-Fold, Leave-One-Subject-Out, and custom split strategies
- **Model Training** — Train EEGNet, SCCNet, and custom deep learning models
- **Evaluation** — Accuracy, AUC, confusion matrix, and saliency map analysis
- **AI Agent** — Natural language interface for guided experiment workflows
